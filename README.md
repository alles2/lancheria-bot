# lancheria-bot
Trabalho da disciplina de Programação Orientada a Objetos II pela UNISC. O objetivo do trabalho é implementar um ChatBot com a API da Telegram, o projeto implementado tem o objetivo de administrar novos pedidos de uma lancheria, onde os pedidos são feitos pelo bot, e salvo no sistema. Além da implementação do Bot, foi implementado o sistema para administrar os pedidos e para adicionar, editar, remover e visualizar os produtos e catagorias dos produtos. Tecnologias utilizadas: Java e MySql.

Especificações do projeto estão no arquivo Especificação-TrabalhoPOOII-2019-2-Segunda.pdf, na raíz do projeto.
